# Zomato AI - Next.js Frontend

A premium, AI-powered restaurant recommendation interface built with Next.js 14, Tailwind CSS, and Framer Motion.

## Prerequisites
- Node.js 18.x or later
- FastAPI Backend (running at http://localhost:8000)

## Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd nextjs_frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features
- **Glassmorphism Design**: Modern, transparent UI elements.
- **Micro-interactions**: Smooth transitions and hover effects using Framer Motion.
- **Dynamic Cards**: Real-time rendering of LLM-generated recommendations.
- **Responsive Layout**: Optimized for both mobile and desktop views.
- **Feedback Loop**: Integrated thumbs-up/down interaction on each recommendation.
